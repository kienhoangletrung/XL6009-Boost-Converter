# XL6009 Boost Converter

DC-DC step-up power module.

## Function
Boosts low DC voltage to higher adjustable output.

## Key Specifications
- Topology: Non-synchronous Boost
- Regulator: XL6009
- Diode: SS510
- Inductor: 33µH
- Current: Up to 4A
